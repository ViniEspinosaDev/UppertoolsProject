Para configurar o Banco:

1º Certifique-se de criar um Banco de Dados chamado "CadastroEmpresa".

2º Posteriormente abra a solução e clique com botão esquerdo em "Database_Configuracao">>Publish>>Edit>>ShowConnection 
e adicione o Server Name, Username, password... Após configurar, clique em Publish para criar as tabelas

3º Abra a solução novamente, vá para "RegraNegocio" >> "Classes" >> "Backwork", abra e cole o caminho do banco de dados como argumento
para a variavel "CaminhoBD"

4º Pronto
